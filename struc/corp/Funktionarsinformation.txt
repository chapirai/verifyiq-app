* identitet -> identity

* representerasAv -> representedBy

* representerasAv.identitet -> representedBy.identity
* insats -> contribution
* anteckning -> note

Rules:

* personnamn and organisationsnamn are mutually exclusive depending on entity type
* funktionarsroller: array of KodKlartext, must preserve both code and text
* identitet.identitetsbeteckning: always string, do not cast to number
* postadress fields may be partially populated depending on domestic vs foreign entity
* representerasAv only present when auditor is a company
* insats only applicable for Kommanditbolag, otherwise null/omitted
* anteckning is free text and optional
* missing optional objects should be null or omitted, not defaulted
