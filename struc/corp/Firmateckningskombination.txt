Rules:

* ordning: numeric ordering field, must not be re-indexed or recalculated
* funktionarsroller: array of KodKlartext objects, structure must be preserved
* funktionarer: array of officer references, nested identity objects must not be flattened
* empty objects in example indicate full nested schema defined elsewhere
* missing arrays should be [] if present but empty, otherwise null/omitted
* this structure is part of a nested array (firmateckning.kodad is array[array]) and must remain nested
