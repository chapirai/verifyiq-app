{
"identitetsbeteckning": "string",
"namnskyddslopnummer": "1",

"AKTIEINFORMATION"
],

Fields:

* identitetsbeteckning: string
* namnskyddslopnummer: string
* organisationInformationsmangd: array[string]
* tidpunkt: string

Mapping:

* identitetsbeteckning -> organizationId

* tidpunkt: optional, must be full timestamp (no milliseconds), round up to nearest second if needed
